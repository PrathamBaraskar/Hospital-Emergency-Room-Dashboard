## 📌 Project Overview
The goal of this project is to build an **Emergency Room Analysis Dashboard in Excel** that helps stakeholders monitor hospital performance, analyze patient trends, and improve operational efficiency.

This dashboard enables healthcare managers to:
- Track emergency room activity
- Identify patient flow patterns
- Monitor waiting times and service efficiency
- Make better data-driven decisions for improving hospital services

---

# 🎯 Project Objectives

The main objectives of this project are:

- Analyze emergency room patient data
- Improve visibility of hospital performance metrics
- Provide actionable insights through an interactive dashboard
- Help stakeholders monitor and manage patient services effectively

---

# 📊 Key Performance Indicators (KPIs)

The dashboard focuses on the following KPIs:

- **Total Number of Patients**
- **Average Waiting Time**
- **Patient Admission Rate**
- **Number of Patients by Age Group**
- **Daily Patient Visits**
- **Department/Service Utilization**
- **Patient Distribution Trends**

These KPIs allow hospital administrators to quickly understand operational efficiency and patient flow.

---

# 🗂 Dataset Preparation

Before creating the dashboard, the data was **cleaned and transformed** to make it suitable for analysis.

### Data Cleaning & Transformation
The following steps were performed:

- Cleaned the dataset
- Handled inconsistent data
- Split the **date and time columns** to extract them separately
- Structured the data to make it analysis-ready

# 📅 Date Table Creation

To support time-based analysis:

- A **new query was created containing dates from**  
  **January 1, 2023 → December 31, 2024**
- A **relationship was established between the date table and the main dataset**

This allows the dashboard to perform **time-series analysis and filtering**.

# 👥 Age Group Calculation

An **Age Group column** was created in **Power Pivot** to categorize patients into different groups.

This enables analysis such as:
- Patient distribution by age
- Identifying which age groups use emergency services most frequently
